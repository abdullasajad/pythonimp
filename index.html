Significance of Problem-Solving Strategies

Topic 3 • Advanced Strategies

Means-Ends Analysis Explained

Means-Ends Analysis (MEA) breaks a large problem into smaller sub-goals and reduces the difference between the current state and goal state step by step.

Example: Road Trip from Trivandrum to Kashmir

Goal: Reach Kashmir from Trivandrum

Current State: At Trivandrum

Sub-goals (Breaking it down):

  1. Drive to Bangalore
  2. Hyderabad
  3. Nagpur
  4. Jhansi
  5. Agra
  6. Chandigarh
  7. Jammu
  8. Srinagar (Kashmir)

Each sub-goal reduces the distance to the final destination. MEA achieves the goal through manageable steps.

Topic 4 • Backtracking

Backtracking / Working Backward Strategy

Backtracking tries possible paths toward a solution. When it reaches a dead end, it reverses steps and tests alternate routes until success.

Example: Maze Problem

If a path reaches a dead end, go back and choose another direction until the goal is found.

Other Examples:

  • Sudoku solver
  • Graph search
  • Forgotten PIN attempts
  • Chess move exploration

Working Backward is similar but starts from the goal and traces steps back to the starting point (e.g., planning arrival time by counting backwards).

Topic 5 • Algorithmic Approach

Algorithmic Approach & Its Advantages

Algorithmic Approach: Uses a step-by-step procedure that guarantees a solution.

Example: Steps to bake a cake or add two numbers.

Advantages

  • Guarantees correct solution if steps are followed
  • Logical and systematic
  • Easy to debug and automate
  • Ensures consistency and accuracy
  • Foundation for writing computer programs
Topic 6 • Process

Steps in Problem-Solving Process

  1. Understand the Problem – Identify inputs, outputs, and requirements
  2. Formulate a Model – Develop the mathematical or logical relationship
  3. Develop an Algorithm – Write step-by-step instructions
  4. Write the Program – Translate algorithm into code (e.g., Python)
  5. Test the Program – Run with different inputs and debug errors
  6. Evaluate the Solution – Check if output meets the requirements

Topic 10 • String Operations

String Slicing in Python

String slicing extracts a portion of a string using index positions.

Syntax: string[start:end:step]

text = "Algorithm"

print(text[0:5])     # Output: Algor
print(text[::2])     # Output: Agri
print(text[::-1])    # Output: mhtirogla (reversed)

Understanding Slicing:

  • [0:5] returns characters from index 0 to 4 (end is exclusive)
  • [::2] returns every 2nd character
  • [::-1] reverses the string

Topic 2 • Pseudocode Example

Pseudocode: Average of Three Numbers

1. Start
2. Input num1, num2, num3
3. sum = num1 + num2 + num3
4. average = sum / 3
5. Print average
6. End

This pseudocode can easily be converted to any programming language.

Topic 3 • Pseudocode Benefits

Advantages of Pseudocode over Natural Language

  • Easier to translate into programming code
  • Avoids ambiguity present in natural language
  • Makes debugging and reviewing logic simpler
  • Language-independent — same pseudocode applies to any programming language
  • Saves time before actual coding

Topic 7 • Conditionals

If-Else Statement in Pseudocode

IF number % 2 == 0 THEN
    Print "Even number"
ELSE
    Print "Odd number"
ENDIF

This checks if a number is divisible by 2 to determine if it's even or odd.

Topic 8 • Flowchart Design

Flowchart: Largest of N Numbers

Logic Steps:

  1. Start
  2. Input N (how many numbers)
  3. Initialize max = 0
  4. Loop through all numbers:
    • If current number > max → update max
  5. Display max
  6. End

Flowchart symbols used: Start (Oval) → Input (Parallelogram) → Loop → Decision (Diamond) → Update max (Rectangle) → Output (Parallelogram) → End (Oval)

Topic 9 • Simple Flowchart

Flowchart: Even or Odd Number

Logic Steps:

  1. Start (Oval)
  2. Input a number (n) (Parallelogram)
  3. Decision: If n % 2 == 0 (Diamond)
    • Yes → Print "Even" (Parallelogram)
    • No → Print "Odd" (Parallelogram)
  4. End (Oval)
Topic 10 • Algorithm Example

Pseudocode: Largest Number in List of 10

1. Start
2. Set max = 0
3. Repeat 10 times:
      Input num
      IF num > max THEN
          max = num
      ENDIF
4. Print "Largest number is", max
5. End

This algorithm compares each number with the current maximum and updates it when a larger number is found.
